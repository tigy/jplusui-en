    • Returns:DomDom.get(id)

      Description

      Selects a single element with the given id attribute.

      This method is extremely efficient for selector, and returns an encapsulated DOM object which can use most Dom funtions of jPlus UI. If the element you need to select is exsist，returns null. This method can short for $.

      Parameters

      idString

      The id of element

      Examples:

      Description:

      Finds the element with the id "a".

      HTML Code:
      <p id="a">once</p> <div><p>two</p></div> <p>three</p>
      JS Code:
      Dom.get("a");
      Returns:
      {<p id="a">once</p>}

    • Returns:DomDom.get(dom)

      Description

      Encapsulate any element into a DOM object. 

      This method returns an encapsulated DOM object which can use most DOM funtions of jPlus UI. If the element you need to select is exsist，returns null. This method can short for $.

      Parameters

      domNode/NodeList/Dom/DomList/Array

      javascript DOM element/elements, encapsulated DOM object/objects, array.

      Examples:

      Description:

      Get an encapsulated DOM object with id "a1"

      HTML Code:
      <p id="a1"></p> <p id="a2"></p>
      JS Code:
      Dom.get(document.getElecmentById('a1')) // the same as Dom.get('a1')
      JS Code:
      Dom.get(['a1', 'a2']); // the same as Dom.get('a1')
      JS Code:
      Dom.get(Dom.get('a1')); // the same as Dom.get('a1')
      Returns:
      {<p id="a1"></p>}

    • Returns:DomListDom.query(doms)

      Description

      Select elements by CSS seletor. 

                Support select all elements by given id, class, tag, also support CSS3 expression, multiple selecor. If no element be found, returns an empty DomList object. This method is short for $$ .

      Parameters

      domsString/NodeList/DomList/Array

      CSS seletor, or javascript DOM elements, encapsulated DOM objects, array

      Examples:

      Description:

      Finds every p element.

      HTML Code:
      <p>one</p> <div><p>two</p></div> <p>three</p>
      JS Code:
      Dom.query("p");
      Returns:
      [  <p>one</p> ,<p>two</p>, <p>three</p>  ]

      Description:

      Finds all p elements，which are the children of div element.

      HTML Code:
      <p>one</p> <div><p>two</p></div> <p>three</p>
      JS Code:
      Dom.query("div > p");
      Returns:
      [ <p>two</p> ]

      Description:

      Finds all input with a attribute "type" is "radio".

      JS Code:
      Dom.query("input[type=radio]");

            Dom.query("#foo\\[bar\\]")
            Dom.query("#foo\\.bar")

    • Returns:StringgetAttr(name, [type])
				
      Description
      Get the value of an attribute for the first element in the set of matched elements.  
      If the element doesn't has such attribute, returns null .
      Parameters
      nameString
      The name of the attribute to get.
      type (optional)Integer
      The type of attribute. if type = 0 , The method returns the value of JavaScript（getAttr("checked", 0) = true）. If the type = 1 , it returns the value of HTML（getAttr("checked", 1) = "checked"）.if type = 2 , returns the value of default（getAttr("checked", 2) = true, No matter the element is checked or not, the value always the same.）. Default to be 0 . 
      Examples:
      Description:
      Find the src attribute of the first img in the page. 
      HTML Code:
      <img id="img" src="test.jpg"/>
      JS Code:
      Dom.get("img").getAttr("src");
